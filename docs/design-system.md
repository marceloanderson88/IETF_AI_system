# Design System: Bussola IETF

## Direcao visual

A Bussola IETF usa uma linguagem de produto operacional: lateral esquerda azul profunda, area principal branca, cards densos, hierarquia clara e sinais coloridos por tipo de acao. O objetivo e parecer uma plataforma confiavel de inteligencia tecnica, nao uma landing page.

## Tokens principais

| Token | Valor | Uso |
|---|---:|---|
| Navy | `#071f44` | Fundo principal da navegacao |
| Blue | `#0f63d8` | Acoes primarias e abas ativas |
| Violet | `#4b3df2` | Inteligencia assistida e destaque |
| Green | `#16a34a` | Confianca alta e estados ativos |
| Orange | `#ea7a12` | Janela de acao e maturacao |
| Red | `#dc2626` | Urgencia e alertas fortes |
| Line | `#dce4f2` | Bordas de cards e inputs |
| Radius | `8px` | Raio padrao da UI |

## Componentes

- Sidebar persistente com marca de bussola, navegacao e perfil.
- Topbar com busca global, seletor PT/ES/EN, notificacoes e usuario.
- Cards operacionais com borda clara, sombra sutil e densidade de informacao.
- Tags para grupos, sinais e topicos.
- Recommendation cards com borda lateral por prioridade.
- Grafo SVG responsivo para adjacencias.
- Formularios de acompanhamento, rascunhos e preferencias.

## Estados

Azul para selecao, verde para confianca, laranja para maturacao, vermelho para urgencia e violeta para inteligencia/recomendacao.

## Acessibilidade

Contraste alto na navegacao, controles com altura minima de 40px, texto principal escuro e estados acompanhados por labels textuais.
